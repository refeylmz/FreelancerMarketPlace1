# FreelancerMarketPlace1

Bu proje, ASP.Net Core 7.0 kullanarak Entity Framework Core ve MVC mimarisi ile kodlanmıştır. Codefirst yaklaşımı ile SQL Server veritabanını destekleyen bir Freelancer sitesi projesidir.
